ゴミ分別ナビ

「これって何ゴミだろう」という日常の小さな迷いを、 すぐ解決できるようにした軽量ナビアプリ。情報の速さと分かりやすさを優先しています。

Summary

この作品で見せたいこと

  • 軽量な技術でも、生活の迷いをすぐ減らせること
  • 説明と導線を整理することで、検索コストを下げられること
  • 公開までの速さと実用性のバランスを取れること

Overview

課題と設計

ゴミの分別は地域差もあり、覚えていても意外と迷いやすいテーマです。 そこで、毎回調べ直すストレスを減らすために、必要な答えへすぐ届く導線を意識して作りました。

重たい構成にはせず、単一ファイルで軽く動くことで、 スマホでもすぐ開ける道具として成立させています。

公開サイト GitHub リポジトリ 代表作一覧に戻る

Highlights

主な機能

  • 入力された品目に対して、分別の考え方を分かりやすく表示
  • HTML と JavaScript を中心にした軽量な単一ファイル構成
  • 公開のハードルを下げ、すぐ使える状態まで持っていくスピード感
  • 暮らしに近い課題をテーマにした、分かりやすい UI 設計

Why It Matters

この作品が示す強み

複雑な機能よりも、「必要なときにすぐ役立つか」を優先して組み立てた作品です。 小さな不便を見つけて、最短距離で解決策に変える姿勢がよく表れています。

こうした軽量プロダクトを早く出せることは、 実務アプリのプロトタイプ設計や MVP 構築にもつながる力だと考えています。
